Mac必备

RVM安装

Install GPG keys:

gpg2 --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

Install RVM:

\curl -sSL https://get.rvm.io | bash -s stable

常用指令：

rvm list known
rvm install 2.x.x
rvm use 2.x.x
rvm use 2.x.x --default

#####Homebrew

安装Homebrew：

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

常用指令：

brew install wget
brew cask install firefox

####Oh-my-zsh

安装：

sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

常用插件：

brew install zsh-syntax-highlighting
brew install zsh-autosuggestions

常用工具

安装iterm2：

brew cask install iterm2

安装git-flow：

brew install git-flow

安装cocoapods：

gem install bundler
bundle install cocoapods
sudo gem update --system // 可能需要更新gem

安装fastlane：

gem install fastlane

安装JDK：

brew cask install homebrew/cask-versions/java8

安装Atom：

brew cask install atom

安装fir-cli：

gem install fir-cli

Mac瘦身

删除不可用模拟器：

xcrun simctl delete unavailable

移除对旧设备的支持：

~/Library/Developer/Xcode/iOS\ DeviceSupport # 路径

移除Archives：

~/Library/Developer/Xcode/Archives # 不可恢复，建议备份DSYM文件夹

移除DerivedData：

~/Library/Developer/Xcode/DerivedData

使用ncdu清理：

brew install ncdu # 安装
ncdu # 启动
sudo ncdu # root权限启动
#  使用enter键进入文件夹；使用d键删除文件或文件夹；使用左右方向键进入和返回文件夹；使用q键退出

清理手动下载的模拟器：

/Library/Developer/CoreSimulator/Profiles/Runtimes
